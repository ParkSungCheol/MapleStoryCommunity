  • 이 프로그램은 2개의 서버를 필요로 합니다.
  • Phaser Live Server Port : 5050 / React Port : 3000
  • 원활한 프로그램 진행을 위해 서버 2개를 미리 구동시켜 놓는 것을 권장합니다.
  • 서버가 모두 동작하면 Start.html을 실행하시면 됩니다.
  • 나머지는 내부적으로 연동되게 처리해 놓았습니다.

Supported Frameworks

phaser 3

react

Installation

- npm

https://nodejs.org/en/download/

- yarn

$ npm install --global yarn

- react-router (페이지를 연결해 주는 라이브러리)

npm react-router

- react-table (정렬해 주는 라이브러리)

npm install react-table --save
yarn add react-table

- semantic-ui (다양한 css 아이콘 코드를 가져오는 라이브러리)

yarn add semantic-ui-react semantic-ui-css

- Live Server

Visual Studio -> Extentions -> Install Live Server
